1. Configurar o Git
git config --global user.name "matheuszaccaro"
git config --global user.email "matheuszacc07@gmail.com"

2. Inicializar o Git
git init

3. Criar o repo no GitHub
nome do repo: Aula01

4. Nesta etapa, é a etapa de desenvolvimento, ou seja, alterações nos nossos arquivos. Site, projetos, arquivos texto, entre outros.

5. Após a conclusão do desenvolvimento, adicionar os arquivos ao repo:
git add .

REPETIR OS PASSOS 6 E 7 SEMPRE QUE HOUVE ALTERAÇÃO NOS ARQUIVOSDO PROJETO

6. Comitar ao repositório com uma mensagem explicativa:
git commit -m "Mensagem entre parênteses"

7. Associar o nosso repo local com o remoto
git remote add origin LINK_DO_REPO
git branch -M main
git push -u origin main
